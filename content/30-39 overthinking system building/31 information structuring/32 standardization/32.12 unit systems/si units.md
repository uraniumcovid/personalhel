one of the cooler unit systems and the one everyone needs to use for practical day-to-day usage.

# base units

| quantity           | unit name | symbol used |
| ------------------ | --------- | ----------- |
| distance           | meter     | m           |
| time               | second    | s           |
| mass               | kilogram  | kg          |
| electrical current | ampere    | A           |
| temperature        | kelvin    | K           |
| light flux         | candela   | cd          |
| amount             | mole      | mol         |
